# ○○社ES作成プロンプト・テンプレート

新しいセッション（Claudeアプリ＋Chrome拡張）で、対象企業のESページを開いた状態で、
下のテンプレの〈　〉を埋めて貼り付ける。これ一発で手順が走る。

---

## テンプレート（コピーして使う）

```
あなたは就活ESの作成支援者。手順は就活ポータルの tools/playbooks/es-browser-session.md に従うこと。
要点: ①開いているESページから設問と文字数上限を全部読む ②ポータル(https://job-hunt-pi-orpin.vercel.app/)の
「資料」タブの自己分析マスター資料・基本プロフィールと「ES作成→設問別」の過去ESを根拠/流用元にする
③だ・である/結論先行/字数9割・超過禁止/事実のみ・無ければ[要確認]/具体定量で下書き
④字数を添えて提示し、私の確認を待つ ⑤確認後ポータルの「ES作成→企業別→対象社→＋ESを追加」で保存し、ステータスも更新。

対象企業: 〈例: NEC〉
ESページ（このブラウザで開いている）: 〈URL〉
コース/インターン名・内容の要点: 〈分かれば。志望動機をここに寄せる〉
特記事項: 〈例: 設問3は研究内容を中心に / 提出締切 〇/〇〉

まず、開いているESページから設問と文字数上限を一覧化して見せて。その後に下書きに進んで。
```

---

## 使い方のコツ
- **設問が多い／長い**ときは、まず「設問の一覧化」だけ先にやらせ、内容を確認してから1問ずつ書かせると精度が上がる。
- **過去に似た設問**がある企業なら、テンプレ末尾に「設問別タブの〈ガクチカ〉を流用ベースに」と足すと早い＝節約。
- 書き上がりは必ず**ポータルに保存**させること（次回の再利用資産になる）。
- このテンプレは「○○社」を変えるだけで全社に使い回せる。
